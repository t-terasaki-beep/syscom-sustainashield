# シスコムサステナシールド LP

建物の資産価値を上げる、省エネ・創エネ ダブルプランのランディングページ。
脂肪族ポリウレアによるトリプルガードコート、窓ガラスコーティング、看板劣化防止塗装、
そして太陽光・蓄電池までをワンストップで提案する事業者・建物オーナー向けLPです。

株式会社シスコムネット 正規代理店として運用するためのテンプレートです。

## GitHub Pagesでの公開手順

1. このリポジトリを GitHub にプッシュ
2. リポジトリ Settings → **Pages** を開く
3. **Source** に `Deploy from a branch` を選択
4. **Branch** を `main` / `(root)` にして Save
5. 数十秒〜数分で `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開

独自ドメインを当てる場合は、Settings → Pages → **Custom domain** に
ドメインを入力し、DNSにCNAME（GitHub Pages宛）を設定してください。

## 主な機能

### 省エネシミュレーター
- 建物用途6パターン（オフィス／飲食／店舗・小売／工場・倉庫／ホテル／病院）から選択
- 毎月／年間の電気代を入力
- 建物用途別の空調比率（経産省・環境省ZEB資料の業界平均）を自動セット
- 遮熱コート＋窓コーティング導入時の **空調25%削減** で年間削減額・10年累計・CO₂削減量を即時試算
  - 業務用電力単価：27円/kWh
  - CO₂排出係数：0.434 kg-CO₂/kWh

### 4つの工法カード
- トリプルガードコート（屋根・外壁の3層構造）
- 脂肪族ポリウレア（20年以上の耐候性）
- 窓ガラスコーティング（UV99%カット／空調25%削減）
- 看板劣化防止塗装（退色・チョーキング抑制）

### CV導線
- 無料相談・現地調査フォーム
- お見積り依頼
- 資料ダウンロード（シスコムネット／脂肪族ポリウレア協議会へリンク）
- 詳細省エネシミュレーション依頼

## 編集ポイント

`index.html` 内のテキストはすべてHTML直書きです。よく更新する箇所：

- 会社情報（フッター、`tel:0432084456` のリンク）
- ヒーローのキャッチコピー（`<h1>` 内）
- シミュレーターの建物データ（`<script>` 内 `BUILDINGS` 配列）
  - 単価や削減率を変えたい場合は `PRICE_PER_KWH` `REDUCTION` `CO2_PER_KWH` を編集
- フォームの送信処理（現状は`alert`デモ。Formspree / Google Forms / 自社CRM等に差し替え）
