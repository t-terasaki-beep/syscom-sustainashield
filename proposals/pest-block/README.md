# 害虫ブロック | 法人向け害虫対策LP

法人向け害虫対策サービス「害虫ブロック」のランディングページ・ワイヤーフレームです。

## ターゲット

- 食品工場
- スーパーマーケット(搬入口・バックヤード)
- ホテル・旅館
- ゴルフ場

## サービスの特徴

- **人体・食品に無害**な薬剤を使用
- 殺虫ではなく「侵入をブロック」するバリア施工
- HACCP対応(施工記録書発行)
- 1,000円/㎡〜

## ファイル構成

- `index.html` — LPワイヤーフレーム本体(1ファイル完結、外部CSSなし)

## 技術スタック

- HTML / CSS only(JS不要)
- Google Fonts: Noto Serif JP / Noto Sans JP / Cormorant Garamond
- レスポンシブ対応(PC・スマホ)
- TERAキャラクターはインラインSVG
